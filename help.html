pywv

Einleitung

Diese Hilfe beschreibt das Projekt pywv.

pvwv ist eine in der Programmiersprache Python erstellte Anwendung.

Es wird eine Landkarte angezeigt, die mit einer einfachen Navigation versehen ist.

Zu der Landkarte kann ein oder mehrere GPX Track(s) geladen und angezeigt werden. Es ist möglich die GPX Tracks zu editieren und abzuspeichern.

Eine weiter Möglichkeit die pywv bietet ist es in OpenStreetMap eingetragene Webseiten anzuzeigen.

Navigation

Die Karte kann auf zwei Art und Weisen bewegt werden. (1) Durch das Menu Position (2) Durch einen Linksklick auf die Karte.

Mit einem Linksklick auf einen Punkt der Karte macht man diesen Punkt zur Mitte der dargestellten Karte.

Das Menu Position hat einige Unterpunkte.

  • Zoom Up : Hier wird die Zoom Stufe um eins erniedrigt
  • Zoom Down : Hier wird die Zoom Stufe um eins erhöht
  • West : Die Karte wird nach Westen bewegt
  • East : Die Karte wird nach Osten bewegt
  • North : Die Karte wird nach Norden bewegt
  • South : Die Karte wird nach Süden bewegt

File Menu

Das File Menu bietet drei Unterpunkte.

  • Print to PDF : Die Karte wird in ein PDF gedruckt
  • Download Tile : Es werden Kacheln nachgeladen
  • Show Websites : Zeigt in OpenStreetMap eingetragene Webseiten an

Dazu noch ein paar Bemerkungen.

Der Name des PDF in das gedruckt wird kann in den Einstellungen vorgenommen werden.

Die Karte besteht aus Kacheln, die von https://www.thunderforest.com/ heruntergeladen werden. Diese Kacheln werden gecacht. Das heißt sobald eine Kachel einmal heruntergeladen wurde befindet sie sich auf der Festplatte und braucht nicht nochmal heruntergeladen zu werden. Ist so eine Kachel fehlerhaft, kann sie mit dem Menu Unterpunkt - Download Tile - nocheinmal nachgeladen werden.

Show Webseites lädt OpenStreetMap Daten in einer kleinen Umgebung vom Kartenmittelpunkt herunter und extrahiert daraus alle eingetragenen Webseiten. Dies werden in einer Liste im Webbrowser anzeigt und können von da aus angezeigt werden.

GPX Menu

Das GPX Menu beinhaltet die Unterpunkte zum Anzeigen, Bearbeiten und Abspeichern von GPX Tracks.

  • Read GPX Track : Liest einen GPX Track ein
  • Save GPX Track as : Speichert den GPX Track ab
  • Set Normal Modus to GPX
  • Set Add Point Modus to GPX
  • Set Delete Rectangle Modus from GPX

Dazu auch hier Bemerkungen.

Ein eingelesener GPX Track wird automatisch zu allen bisher eingelesenen und bearbeiteten GPX Tracks hinzugefügt. Bereits vorhandene GPX Tracks werden nicht überschrieben.

Beim abspeichern eines GPX Tracks gehen dessen Meta Informationen verloren. Lediglich die Track Points werden abgespeichert.

Set Normal Modus to GPX - Beendet das Editieren des GPX Tracks. Dieser wird nun nur angezeigt.

Set Add Point Modus to GPX - Ab jetzt wird durch jeden Rechtsklick auf die Landkarte der zugehörige Punkt dem GPX Track hinzugefügt. Der Punkt bekommt die aktuelle Uhrzeit und die Höhe 0.0.

Set Delete Rectangle Modus - Ab jetzt haben die beiden folgenden Rechtsklicks auf die Karte eine Bedeutung. Der erste Rechtsklick setzt eine Ecke eines Rechtecks. Der zweite die entgegengesetzte Ecke des Rechtecks. Damit ist ein Rechteck ausgewählt. Alle Track Points des GPX Tracks, die sich in diesem Rechteck befinden werden gelöscht. Das Editieren des GPX Tracks wird anschließend beendet.

Einstellungen

Alle Einstellungen werden über eine ini Datei vorgenommen. run.ini unter Windows und runLinux.ini unter Linux.

Die ini Datei besteht aus einzelnen Zeilen. Das Format einer Zeile ist: ParameterName=ParameterWert=. Dabei gibt es folgende ParameterNamen:

  • temphtml
  • osmxml
  • pdfFile
  • tileCache
  • x
  • y
  • z
  • adjustment
  • apiKey

Die Parameter werden nun im Folgenden beschrieben

temphtml - Ist der Name einer Datei in dem pywv temporär Links zu Webseiten abspeichert.

osmxml - Ebenfalls ein Dateiname. Hier werden von OpenStreetMap heruntergeladene XML Daten gespeichert.

pdfFile - Das PDF in dem eine Karte als PDF gedruckt werden kann.

tileCache - Eine wichtige Einstellung. Es wird ein Ordner angegeben in dem pywv heruntergeladene Kachel zwischenspeichert. Dieser Ordner kann im Laufe der Zeit eine große Menge an Dateien beinhalten.

x,y,z - gibt den Ort an, der sich zu Anfang der Darstellung im Kartenmittelpunkt befindet.

adjustment - Interne Einstellung sollte normalerweise nicht verändert werden.

apiKey - Um Kacheln von Thunderforest herunterzuladen benötigt man einen apikey. Der per Default eingetragene apikey ist nur ein Test apikey. Das bedeutet mit ihm können maximal 150000 Kachel pro Monat heruntergeladen werden. Wenn Sie gefallen an pywv finden und Sie diesen häufig nutzen, bitte ich Sie sich einen eigenen apikey zu besorgen.
